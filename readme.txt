Simple meta tags generator to improve sharing appearance accross search engines and social media.

== Description ==

Meta tags IO generator is an easy to use plugin which serves as one-stop solution to meta tags. You can simply add meta image, site title, description using the dedicated dashboard and the meta tags are auto generated to the &lt;head>.

There other features as well, such as WooCommerce integration to rock your products in social media, independent posts/pages with their own meta such as Post title, post featured image etc.

Great thanks to <strong>Sergey Babayan</strong>, <a href="https://www.prospectacy.com/author/sergey/" target="_blank">Prospectacy Operations Manager</a> for his great feedback to get this done.

Dashboard page:
Settings -> Meta Tags

== Frequently Asked Questions ==

= Are there any settings? =

Yes, there are is a dedicated dashboard page in WP admin panel in which you can do everything!

= How can I make my WP posts have their own meta tags? =

Through the dedicated dashboard page, simple check the radio button 1->Use featured image for WP Posts & 2->Use teaser/excerpt for WP Posts

= How does WooCommerce works? =

You need the integrator plugin to light up your products in social media in a way no one has ever done before.
